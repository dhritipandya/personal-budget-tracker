# Personal Budget Tracker

Student: Dhriti Pandya
Course: Software Engineering

# Project Description

This is a web-based personal budget tracker that allows users to record income and expenses, view a financial summary, and track transactions. The application is designed to help users manage their personal finances easily and efficiently.

# Features (Week 2)

-Add transactions (income or expense) with title, amount, date, category, and type
-View transactions in a dynamic table
-Automatic calculation and display of total income, total expenses, and net balance
-Validation to ensure required fields are completed and amount is numeric

Note: Full edit and delete functionality (CRUD) will be implemented in Week 3.

# Technologies Used

HTML
CSS
JavaScript

# Week 2 Progress

-Implemented transaction entry form with field validation
-Transactions are stored in a JavaScript array for session-based data management
-Dynamic table displays all added transactions
-Dashboard updates financial summary automatically whenever a transaction is added

# How to Run

-Clone or download the repository.
-Open index.html in a web browser.
-Use the form to add income or expense transactions and view the summary.
